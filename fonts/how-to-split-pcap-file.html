How to Split PCAP Files

May 5, 2016, 11:08pm
Published

Like: Dislike:

When you have a large PCAP file sometimes you'll need to split it up into multiple smaller versions by using some variable. In this example I have a 4GB PCAP file that i'll be splitting up based on the connection flows between the hosts.

You'll need a free application called SplitCap which you can find at https://www.netresec.com/?page=SplitCap.

Using the terminal, launch the tool and pass in the arguments: SplitCap.exe -r <file> -s <split type>. For the -s flag you can pass in the following arguments:


  • flow

  • host

  • hostpair

  • nosplit

  • session

  • seconds <n>

  • packets <n>



You can also get more details about these commands by simply executing the program with no commands.

After you run the program with the flow tag for example, you'll get a folder of new PCAP files.
